## Requirements

The theme "loveit" is version 0.3.0. This does not appear to fully work on Hugo versions > 0.145.
Therefore, be sure to install hugo version 0.145:

```sh
wget https://github.com/gohugoio/hugo/releases/download/v0.145.0/hugo_0.145.0_linux-amd64.deb
sudo dpkg -i hugo_0.145.0_linux-amd64.deb 
```
